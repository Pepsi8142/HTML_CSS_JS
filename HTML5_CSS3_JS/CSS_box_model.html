Box Model Calculation

Total width taken by a content is explained below with calculation formula and some css properties.


Total Width = WIDTH of paragraph content + MARGIN LEFT + MARGIN RIGHT + PADDING LEFT + PADDING RIGHT + BORDER LEFT + BORDER RIGHT


Same process will be applicable for Height.

N.B: This process will be complicated in real life projects. This method will be replaced by BOX SIZING concept.
